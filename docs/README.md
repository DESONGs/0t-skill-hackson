## 清理原则

以下类型的旧文档已经或应当被清理：

- 已经失效的计划态里程碑文档
- 与当前实现重复、但内容更旧的合同草稿
- 只适用于初始搭建阶段的状态文档

后续新增文档时，优先遵守两条规则：

- 只记录当前仍有效的实现与运维信息
- 不保留与现状冲突的历史计划副本
